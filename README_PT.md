# Norway Olympic Predictor

## Visão Geral

O **Norway Olympic Predictor** é uma solução avançada de Machine Learning projetada para prever o número de medalhas que a Noruega deve conquistar em futuros Jogos Olímpicos. Ao analisar dados históricos de desempenho olímpico em conjunto com indicadores socioeconômicos-chave — como PIB, IDH e população — o modelo utiliza o algoritmo Random Forest para gerar previsões altamente precisas (score R² de 0,94).

## Fundamentação Técnica

O modelo é construído sobre uma pipeline de dados robusta:

*   **Dados Históricos:** Conjuntos abrangentes de contagem de medalhas da Noruega e demais nações.
*   **Indicadores Socioeconômicos:** Integração de estatísticas de PIB, Índice de Desenvolvimento Humano (IDH) e população.
*   **Algoritmo de Machine Learning:** Um regressor Random Forest afimado para precisão preditiva.
*   **Processamento de Dados:** Técnicas avançadas de pré-processamento para tratamento de outliers, engenharia de features e normalização.
