### `requirements.txt`

Este arquivo lista todas as dependências necessárias para executar o projeto. Para instalar todas as dependências, utilize o comando `pip install -r requirements.txt`.

```plaintext
pandas==1.5.3
numpy==1.23.5
scikit-learn==1.1.3
matplotlib==3.6.2
xgboost==1.7.1
```

### Como usar

1. Salve o conteúdo acima em um arquivo chamado `requirements.txt`.
2. No terminal, navegue até o diretório do projeto e execute:

   ```bash
   pip install -r requirements.txt
   ```

Isso irá instalar todas as bibliotecas necessárias nas versões especificadas, garantindo que o projeto funcione conforme esperado.
